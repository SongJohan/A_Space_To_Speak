# 🌌 A Space To Speak
**Winner: Best Beginner Project @ StormHacks 2024**

A place to hear and be heard in the vastness of our universe.

## 💡 Inspiration
Many youths struggle with mental health but face significant stigma when seeking help. **A Space To Speak** provides an anonymous sanctuary to express thoughts and feelings, connecting users with others to remind them they aren't alone.

## 🚀 Features
* **Star-Bound Thoughts:** Send your voice into a digital universe.
* **Global Connection:** View thoughts posted by others to find comfort in shared experiences.
* **Anonymity:** A judgment-free outlet designed for those with nowhere else to turn.

## 🛠️ Built With
* **Frontend:** Java (Android Studio)
* **Backend:** Firebase (Authentication & Real-time Database)
* **Tools:** GitHub Copilot, Android Animations API
